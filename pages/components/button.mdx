
# Figuro-UI: Button Component Showcase

Welcome to the Figuro-UI Button component showcase! Below, you'll find a detailed example of how to use the `Button` component, including its various variants. This template will help you understand how to integrate and customize the button in your Next.js application.

## Button Component

### Basic Usage

To use the `Button` component, first ensure that you have Shadcn for Next.js installed. You can then import the button and use it in your React components.

**Import the Button Component:**

```jsx
import { Button } from 'figuro-ui';
```

**Basic Button Example:**

```jsx
<Button>Click Me</Button>
```

This code renders a default button with the text "Click Me".

### Button Variants

The `Button` component comes with several variants to fit different styles and use cases. Below are examples of each variant.

#### 1. **Primary Button**

**Usage:**

```jsx
<Button variant="primary">Primary Button</Button>
```

**Appearance:**

This button has a bold background color, making it ideal for primary actions.

#### 2. **Secondary Button**

**Usage:**

```jsx
<Button variant="secondary">Secondary Button</Button>
```

**Appearance:**

This button features a more subdued background color, suitable for secondary actions.

#### 3. **Danger Button**

**Usage:**

```jsx
<Button variant="danger">Danger Button</Button>
```

**Appearance:**

With a striking color, this button is perfect for actions that require attention or may result in changes.

#### 4. **Disabled Button**

**Usage:**

```jsx
<Button variant="primary" disabled>Disabled Button</Button>
```

**Appearance:**

This button is grayed out and cannot be interacted with, ideal for indicating unavailable actions.

### Button Sizes

The `Button` component also supports different sizes:

#### **Small Button**

**Usage:**

```jsx
<Button size="small">Small Button</Button>
```

**Appearance:**

A compact button for less prominent actions.

#### **Medium Button**

**Usage:**

```jsx
<Button size="medium">Medium Button</Button>
```

**Appearance:**

The default size for most buttons.

#### **Large Button**

**Usage:**

```jsx
<Button size="large">Large Button</Button>
```

**Appearance:**

A larger button for emphasis or to attract attention.

### Full Example

Here’s a complete example demonstrating all the variants and sizes together:

```jsx
import { Button } from 'figuro-ui';

const ButtonShowcase = () => (
  <div>
    <h2>Button Variants</h2>
    <Button variant="primary">Primary Button</Button>
    <Button variant="secondary">Secondary Button</Button>
    <Button variant="danger">Danger Button</Button>
    <Button variant="primary" disabled>Disabled Button</Button>
    
    <h2>Button Sizes</h2>
    <Button size="small">Small Button</Button>
    <Button size="medium">Medium Button</Button>
    <Button size="large">Large Button</Button>
  </div>
);

export default ButtonShowcase;
```
